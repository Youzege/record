#### 引用类型转原始类型

[] + [] = ''

本质：数组转义后为空字符串

`[].toString() = ''`



[] + {} = '[object Object]'

本质：对象转义后为'[object Object]'



{} + [] = 0

本质：{}; + []，对象{}为一个代码块包裹着，+ []后 对象隐式转换为空字符串，然后进行加法转成数字类型，0



{} + {} = '[object Object] [object Object]'

本质是：在chrome内核浏览器中，会被转成({} + {})，因为chrome会将以左大括符开头，右大括符结尾的代码块用()小括号包裹起来，而其他浏览器会返回 NaN
